# Portal RedMobile

Portal de representantes comerciais da RedMobile para lançamento e acompanhamento de pedidos de venda.

## O que é

O Portal RedMobile é a ferramenta utilizada pelos representantes comerciais da Red para interagir com os clientes varejistas que revendem os produtos da empresa. Por meio do portal, os representantes conseguem registrar pedidos, consultar orçamentos, acompanhar faturamento, comissões, metas e títulos em aberto.

Toda a lógica de negócio e os dados do sistema residem no servidor Protheus — o portal funciona como uma interface de acesso a essas informações, exibindo-as de forma organizada e facilitando o trabalho dos representantes no dia a dia.

## Tecnologias

O projeto é composto por um back-end em .NET e um front-end em Vue 3, ambos containerizados e publicados em conjunto via Docker Compose.
